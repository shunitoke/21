Политика конфиденциальности

Программа 21 — Приватный трекер привычек

Дата вступления в силу: 1 февраля 2026 г.

1. Общие положения

Настоящая Политика конфиденциальности описывает, как приложение "Программа 21" обрабатывает данные пользователей.

2. Принцип приватности

Все данные хранятся локально на устройстве пользователя. Приложение не передаёт персональные данные на серверы разработчика или третьих лиц.

3. Какие данные собираются

  • Названия привычек (вводятся пользователем)
  • История выполнения привычек
  • Записи в журнале практики
  • Настройки приложения (тема, язык)

4. Где хранятся данные

Все данные хранятся в локальной базе данных IndexedDB на устройстве пользователя. Данные не синхронизируются с облаком автоматически.

5. Передача данных третьим лицам

Приложение не передаёт данные третьим лицам. Исключение составляют только:

  • Функция экспорта/импорта данных (по инициативе пользователя)
  • Стандартные сервисы операционной системы Android

6. Права пользователя

Пользователь имеет право:

  • Экспортировать все свои данные через настройки приложения
  • Удалить все данные через "Сбросить все данные" в настройках
  • Удалить приложение — при этом все данные автоматически удаляются

7. Контакты

По вопросам, связанным с конфиденциальностью, обращайтесь: support@programma21.app
